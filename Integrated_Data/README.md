This folder contains the data integration workflow, utilizing the record linkage package. The two datasets were first blocked by the state variable, thus reducing the number of comparisons required in the record linkage process. We compared the two datasets on the institution name variable, using the levenshtein method as our metric with a threshold of 0.85. Following the integration process, we dropped any duplicate institution rows and used the admissions dataset as our main reference for duplication entries / conflicting results.
